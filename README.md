# Retail Sales Analysis – Tableau Project

This project explores retail sales trends using Tableau, based on the M5 Forecasting – Accuracy dataset.  
It focuses on understanding store performance, product demand, and event-driven seasonality across three years (2014–2016).


## Dashboards Overview

### Dashboard 1 – Retail Overview
- Shows overall sales 
- Highlights top-performing stores & categories.

### Dashboard 2 – Store Performance
- Compares store-wise sales 
  
### Dashboard 3 – Seasonality & Events
- Displays weekly and monthly seasonality.
- Reveals top events influencing sales trends.



## Key Insights
- **CA_1** emerged as the top-performing store.
- Sales peak during **October** with strong event influence.
- **Weekly seasonality** shows recurring weekend sales boosts.



## Tools Used
- Tableau Desktop  
- Excel / CSV dataset (M5 Forecasting – Accuracy)  
- GitHub for project documentation
